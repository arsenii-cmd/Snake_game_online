🚀 БЫСТРЫЙ СТАРТ
═══════════════════════════════════════════════════════════════

1️⃣ УСТАНОВКА
   pip install -r requirements.txt
   npm install

2️⃣ НАСТРОЙКА
   cp .env.example .env

3️⃣ ЗАПУСК СЕРВЕРА
   python server.py

4️⃣ ЗАПУСК КЛИЕНТА (новый терминал)
   npm run dev

5️⃣ ОТКРЫТЬ БРАУЗЕР
   http://localhost:3000


⚙️ КОНФИГУРАЦИЯ ЧЕРЕЗ .ENV
═══════════════════════════════════════════════════════════════

Сервер:
  SERVER_HOST=localhost      # Хост сервера
  SERVER_PORT=8765          # Порт WebSocket

Клиент:
  VITE_PORT=3000            # Порт веб-приложения
  VITE_WS_URL=ws://localhost:8765  # URL WebSocket

📡 ПРОТОКОЛ WEBSOCKET
═══════════════════════════════════════════════════════════════

Типы сообщений:
  • create_room      - Создание комнаты
  • join_room        - Присоединение к комнате
  • start_game       - Начало игры
  • game_state       - Синхронизация состояния
  • player_input     - Ввод игрока
  • game_over        - Окончание игры
  • player_joined    - Игрок присоединился
  • player_left      - Игрок вышел
  • error            - Ошибка
