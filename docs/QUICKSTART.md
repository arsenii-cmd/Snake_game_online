# Быстрый старт - Мультиплеер

## Шаг 1: Запуск сервера

```bash
python server.py
```

Вы должны увидеть:
```
Запуск сервера на ws://localhost:8765
```

## Шаг 2: Тестирование сервера (опционально)

### Способ 1: Python скрипт
В новом терминале:
```bash
python test_server.py
```

Если все работает, вы увидите:
```
✓ Подключено к серверу
✓ Успешно присоединились к игре
✓ Тест пройден успешно!
```

### Способ 2: HTML страница
Откройте `test_connection.html` в браузере и нажмите "Подключиться"

## Шаг 3: Запуск игры

1. Откройте `index.html` в браузере
2. Выберите "Мультиплеер"
3. Убедитесь, что адрес сервера: `ws://localhost:8765`
4. Нажмите "Настроить управление"
5. Выберите управление (по умолчанию WASD/ЦФЫВ)
6. Нажмите "Начать игру"

## Шаг 4: Подключение второго игрока

1. Откройте `index.html` в другой вкладке или на другом устройстве
2. Повторите шаги из Шага 3

## Игра через интернет

### На сервере:

1. Откройте `server.py`
2. Измените `host = "localhost"` на `host = "0.0.0.0"`
3. Запустите сервер
4. Узнайте ваш внешний IP (например, через https://whatismyip.com)

### На клиенте:

1. Откройте игру
2. Выберите "Мультиплеер"
3. Введите адрес: `ws://ВАШ_IP:8765`
4. Начните игру

## Управление

- **WASD** / **ЦФЫВ** - Игрок 1
- **Стрелки** - Игрок 2 (в локальной игре)
- **Esc** / **ё** - Пауза
- **C** - Полноэкранный режим

## Решение проблем

### Сервер не запускается
```bash
pip install websockets
```

### Не удается подключиться
- Проверьте, что сервер запущен
- Проверьте адрес сервера
- Проверьте файервол

### Игра лагает
- Уменьшите количество игроков
- Проверьте скорость интернета
- Используйте локальную сеть вместо интернета
