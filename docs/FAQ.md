# Часто задаваемые вопросы (FAQ)

## Общие вопросы

**Q: Сколько игроков может играть одновременно?**
A: Локальная игра - до 4 игроков + 7 ботов. Онлайн мультиплеер - неограниченно (рекомендуется до 8).

**Q: Нужен ли интернет для игры?**
A: Для локальной игры - нет. Для онлайн мультиплеера - да.

**Q: На каких устройствах можно играть?**
A: На любых устройствах с современным браузером (компьютеры, планшеты, смартфоны).

## Технические вопросы

**Q: Ошибка "websockets module not found"**
A: Установите зависимости: `pip install -r requirements.txt`

**Q: Ошибка "Address already in use"**
A: Порт 8765 занят. Остановите другой процесс или измените порт в server.py

**Q: Не могу подключиться к серверу**
A: Проверьте что сервер запущен, адрес правильный, файервол не блокирует порт.

## Игровые вопросы

**Q: Как работают боты?**
A: Боты имеют 5 стратегий: агрессивный, осторожный, случайный, территориальный, охотник.

**Q: Что такое режим "Призрак"?**
A: Режим где змейки могут становиться невидимыми с разными вариантами.

**Q: Как стрелять в режиме "Магический шутер"?**
A: Стрельба автоматическая при эффекте "Оружие".
