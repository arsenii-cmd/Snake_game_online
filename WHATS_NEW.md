# Что нового в версии 2.0

## Главная новинка: Онлайн мультиплеер!

Теперь вы можете играть с друзьями через интернет в реальном времени!

## Новые возможности

### Мультиплеер
- Игра через интернет или локальную сеть
- Неограниченное количество игроков
- Синхронизация в реальном времени
- Поддержка всех режимов игры

### Интерфейс
- Новое меню выбора режима (Локальная игра / Мультиплеер)
- Поле ввода адреса сервера
- Индикатор подключения
- Улучшенная обратная связь

### Тестирование
- Python скрипт для тестирования сервера
- HTML страница для диагностики подключения
- Подробные логи в реальном времени

## Улучшения

### Производительность
- Оптимизирована отправка данных
- Уменьшена задержка синхронизации
- Улучшена обработка отключений

### Совместимость
- Полная обратная совместимость с локальной игрой

## Как попробовать

1. Запустите сервер: `python server.py`
2. Откройте игру в браузере
3. Выберите "Мультиплеер"
4. Начните игру!

Подробнее в QUICKSTART.md
