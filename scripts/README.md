# Скрипты запуска

## macOS/Linux

```bash
./START.sh
```

## Windows

### Локальный запуск
```cmd
START_LOCAL.bat
```

### Публичный доступ
```cmd
START_PUBLIC.bat
START_PUBLIC_AUTO.bat
```

### PowerShell
```powershell
.\start_public.ps1
```

### Остановка серверов
```cmd
STOP_ALL_SERVERS.bat
```

### Проверка файрвола
```cmd
CHECK_FIREWALL.bat
```
